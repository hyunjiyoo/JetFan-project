## 제트팬 관리 시스템
제트팬 관리 시스템은 터널을 환기시켜주기 위해 설치된 제트팬의 상태를 관리해주는 안드로이드용 웹앱입니다.

### 프로젝트 기간
- 2020.11.25 ~ 2021.02.23

### 주 사용기술
- 프론트엔드(유현지): javascript, python, jinja2, less
- 백엔드: python, swagger, mysql, docker
